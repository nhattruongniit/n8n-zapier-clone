## n8n zapier clone with Next.js 13, Tailwind CSS, and TypeScript

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- nextjs, shadcn/ui, tailwindcss, typescript
- db: neon
- payment & subscription
- authenticate
- workflow execution realtime update
- visual canvas react flow
- AI integration: openai, gemini
- code rabbit
- sentry
